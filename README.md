# Лендинг выбора тура путешествий по Японии:
## Этапы разработки:
* Разметка страницы с помощью тегов HTML;
* Добавление стилей к верстке с помощью таблиц стилей (CSS);
* Добавление интерактивности странице лендинга с помощью языка JavaScript;
* Разработка адаптивного дизайна под iPad и мобильные устройства;
* Оптимизация файлов и кода;
* Загрузка лендинга в систему контроля версий (GitHub).
